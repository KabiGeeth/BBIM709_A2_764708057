This is a big data analytics portfolio built in Databricks. It covers Map Reduce, ETL pipeline and Machine Learning on Uber Ride Booking data. 
The dataset is from Kaggle, named "Uber Ride Analytics 2024 (NCR ride bookings). https://www.kaggle.com/datasets/yashdevladdha/uber-ride-analytics-dashboard
The dataset has 150000 records and 21 columns
I have used Databricks Free edition (serverless) and applied PtSpark, Delta lake, Spark MLlib, and Matplotlib
